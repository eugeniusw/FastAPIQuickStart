# FastAPI QuickStart
Provides scalable FastAPI project structure

# Usage
* clone this repo
* install FastAPI
    ```bash
    pip install "fastapi[all]"
    ```
* run
    * using uvicorn
        ```
        uvicorn main:app --relaod
        ```
    * using python
        ```
        python main.py
        ```
* browse the sample app http://localhost:8000/v1/hello
